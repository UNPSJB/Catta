    • Pantalla Principal


    La pantalla principal del dueño se divide en tres zonas donde se muestra prácticamente la funcionalidad completa del sistema.

    Menú Principal (1) donde se muestran los botones:

    • Principal: Permite redirigirse a la pantalla principal.
    • Empleados: Listado de los empleados, Modificar Porcentaje de Comisión.
    • Clientes: Listado de los clientes.
    • Servicios: Listado de los servicios.
    • Insumos: Listado de insumos. Modificar stock de insumo.
    • Turnos: Listado de turnos. Modificar Turno. Confirmar Turno. Realizar Turno. Cancelar Turno.
    • Comisiones: Listado de las comisiones realizadas a los empleados.
    • Ayuda: Accede a la documentación de ayuda de manera contextual.
    • Calendario: Muestra gráficamente un calendario con los turnos correspondientes a cada día y horario.
    • Cuenta: Permite acceder a la opción de cerrar sesión.

    Menú Lateral (2) muestra algunas de las funcionalidades del sistema más utilizadas por el dueño:

    • Crear Turno: Permite crear un nuevo turno.
    • Crear Servicio: Permite crear un nuevo servicio.
    • Crear Promoción: Permite crear una nueva promoción.
    • Crear Empleado: Permite crear un nuevo empleado.
    • Crear Cliente: Permite crear un nuevo cliente.
    • Crear Sector: Permite crear un nuevo sector.
    • Crear Insumo: Permite crear un nuevo insumo.
    • Liquidar comisión: Permite liquidar una comisión para cada empleado.
    • Crear Turno Fijo: Permite crear una serie de turnos fijos.
